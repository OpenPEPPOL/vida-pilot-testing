# NW-HP-002-IC

Files `NW-HP-002-IC.C1-C2.PeppolBIS.xml` implement a basic testing scenario for an **Intra-Community Supply (ICS)** of services, where the VAT Category used is "K" with VAT rate indicated as 0% because the tax responsibility is shifted to the buyer.

> Note: This is a variation of the test scenario `NW-HP-002`, which uses `AE` category to specifically refer to "Reverse Charge" VAT handling.  
> Whether using `K` or `AE` should be used is subject to further clarification.  
> See also this informational document for [Technical guideance for tax codes in EN 16931](https://ec.europa.eu/digital-building-blocks/sites/download/attachments/467108974/eInvoicing%20technical%20guidance%20document_v1.pdf?version=2&modificationDate=1721117125403&api=v2)
