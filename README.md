
# Peppol ViDA Pilot Testing

## Introduction

This repository is designed to provide invoices as test files for the Peppol ViDA Pilot Testing phase.  
It is intended to be used by the Peppol ViDA Pilot participants for testing and validation within the Peppol ViDA Pilot environment.


## Key Features
- **Invoices in Multiple Formats**: The repository contains invoices in multiple formats:
  - XML (PEPPOL BIS Billing 3.0) as the leading data source
  - PDF as a human-readable format for easy review.
- **Detailed Documentation**: Each invoice is annotated with detailed metadata to ensure comprehensive testing across different systems and processes.
- **Supporting files**
   - Validation reports
   - Secondary XML formats: EN16931 CII and EN16931 UBL


## Structur
- **One directory for each test**: Contains the testing files for a specific test case.
- **Naming convention**: The test directory names correspond to the code defined in the Peppol ViDA Pilot Methodology and Testing document available in the [Peppol ViDA Pilot Wiki](https://openpeppol.atlassian.net/wiki/spaces/VP)

## Usage
To use this repository for Peppol ViDA Pilot Testing:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/OpenPeppol/vida-pilot-testing.git
   ```
2. **Access Test Files**: Navigate to the `/[test scenario]` directory and find the invoices in various formats.

3. **Integrate with Your Project**: Use these test files to integrate into your systems for testing and validation.
